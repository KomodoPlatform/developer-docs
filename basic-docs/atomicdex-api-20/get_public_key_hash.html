# get_public_key_hash

The get_public_key_hash method returns the RIPEMD-160 (opens new window) hash version of your public key

# Arguments

Structure Type Description
(none)

# Response

Structure Type Description
public_key_hash string User's RIPEMD-160 public key hash

# üìå Examples

# Command

curl --url "http://127.0.0.1:7783" --data "
{
     \"userpass\": \"${userpass}\",
     \"mmrpc\": \"2.0\",
     \"method\": \"get_public_key_hash\",
     \"params\": {},
     \"id\": 0
}
"

# Response (success)

{
  "mmrpc": "2.0",
  "result": {
    "public_key_hash": "05aab5342166f8594baf17a7d9bef5d567443327"
  },
  "id": 0
}
