# version

**version()**

The `version` method returns the Komodo DeFi Framework version.

#### Arguments

| Structure | Type | Description |
| --------- | ---- | ----------- |
| (none)    |      |             |

#### Response

| Structure | Type   | Description                       |
| --------- | ------ | --------------------------------- |
| result    | string | the Komodo DeFi Framework version |

#### :pushpin: Examples

#### Command

```bash
curl --url "http://127.0.0.1:7783" --data "{
  \"method\":\"version\",
  \"userpass\":\"$userpass\"
}"
```
